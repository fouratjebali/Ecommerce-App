# GreenCraft Marketplace

Sprint 1 foundation for a sustainable handmade goods marketplace inspired by editorial artisan storefronts and built with Angular, NestJS, PostgreSQL, and Docker Compose.

## Workspace

- `apps/web`: Angular storefront shell
- `apps/api`: NestJS commerce API (coming in the next branch)
- `docs`: Architecture, domain, and API design notes
- `infra`: Local development infrastructure

## Current focus

This branch establishes the workspace bootstrap and frontend scaffold so the rest of Sprint 1 can layer in backend services, infrastructure, and architecture documentation.
