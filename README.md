# GreenCraft Marketplace

Sprint 1 foundation for a sustainable handmade goods marketplace inspired by editorial artisan storefronts and built with Angular, NestJS, PostgreSQL, and Docker Compose.

## Workspace

- `apps/web`: Angular storefront shell
- `apps/api`: NestJS commerce API
- `docs`: Architecture, domain, and API design notes
- `infra`: Local development infrastructure

## Current focus

Sprint 1 is establishing the shared foundations for the marketplace: frontend shell, backend modules, local infrastructure, and architecture-ready documentation.

## Quick start

1. Copy `.env.example` to `.env`
2. Run `npm run infra:up`
3. Run `npm run dev:api`
4. Run `npm run dev:web`
